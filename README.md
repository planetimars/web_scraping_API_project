# Web Scraping and API Integration Project

## Project Description
Python project that combines web scraping from Telegrafi.com with API integration from Open-Meteo.

## Features
- Web scraping of news articles from Telegrafi.com
- Weather data retrieval using Open-Meteo API
- JSON data storage and processing
- Error handling and data validation

## Files Included
- `scraping_telegrafi.py` - Web scraping from Telegrafi.com
- `scraping_telegrafi_fixed.py` - Improved version with date handling
- `api_open_meteo.py` - Weather API integration
- `telegrafi_lajmet.json` - Scraped news data
- `moti_tirane.json` - Weather data for Tirana
